# 好友接口

### 1 好友列表接口

> http-get ```/api/friends```

##### 1.1 请求参数说明
|参数|名称|值类型|是否可空|备注
|---|---|---|---|---|
| page |第几页|int|否|传递的页数|
| per_page |每页的条数|int|是|每页的数据条数（默认10）|
| type |状态|int|是|1 好友列表 2 新的好友列表（默认1）|

##### 1.2 回应参数说明
```
{
    error_code  0 成功，非0失败
    users  用户信息数组
}
```
